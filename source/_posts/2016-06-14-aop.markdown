```
以python为例，装饰器（decorator）就是个面向切面编程的好用的工具
比如我现在有很多函数，我想给每个函数加个日志功能，在每个函数调用前都能输出
calling fun.__name__
那么我难道要给每个函数都加条print吗？用装饰器就可以解决这个问题
def log(func):
    def wrapper(*args, **kw):
        print('call %s():' % func.__name__)
        return func(*args, **kw)
    return wrapper
这样，我只需要在每个函数定义前加一行
@log 
def now():
    print('2016-6-14')
这样就相当于log(now())了
```
